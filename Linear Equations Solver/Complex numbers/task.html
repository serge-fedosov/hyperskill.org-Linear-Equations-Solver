Description

In this stage, you should implement the ability to work with complex numbers. A set of complex numbers is wider than a set of real numbers. You can look here to see how to implement addition, subtraction, multiplication, and division with complex numbers.

You can read a complex number in the following way: a double number, then a plus or a minus, then another double number, and then the symbol i. If there is no plus or minus, then the number has the imaginary part equal to zero. If there is no plus or minus but the number ends with i, then the number has the real part equal to zero. Also, if there is 1 before i, like in 2+1i or -1i, then 1 can be omitted, like in 2+i and -i.

Remember that you should write to file either No solutions, or Infinitely many solutions, or the solution. Every variable of the solution should be on a separate line and shouldn't contain any spaces (real part, plus/minus, imaginary part).

If the real part of a number is equal to zero, you should write only the imaginary part, and in case the imaginary part also equals zero, then you should simply write 0. Also, if the imaginary part is equal to zero, then write only the real part. If the imaginary part is equal to 1, write i instead of 1i, like in the parsing examples above.

Example

Suppose you have a file named in.txt, which contains the following (note that the first line contains the number of rows and the number of columns):

3 3
1+2i -1.5-1.1i 2.12 91+5i
-1+3i 1.2+3.5i -3.3 1+15i
12.31 1.3-5i 12.3i -78.3i

Below is how your program might work. The lines which start with > represent the user input.

> java Solver -in in.txt -out out.txt
Start solving the equation.
Rows manipulation:
R1 / 1+2i -> R1
1-3i * R1 + R2 -> R2
-12.31 * R1 + R3 -> R3
R2 / 1.6+6.1i -> R2
-10.40+9.67i * R2 + R3 -> R3
R3 / -6.78+9.71i -> R3
0.54-0.74i * R3 + R2 -> R2
-0.42+0.84i * R3 + R1 -> R1
0.74-0.38i * R2 + R1 -> R1
The solution is: (6.73-22.99i, -1.79+2.0i, 15.6994+7.396i)
Saved to file out.txt

And the file out.txt should look like this: 

6.73-22.99i
-1.79+2.0i
15.6994+7.396i
